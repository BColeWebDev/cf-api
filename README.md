# cf-api
Crown Fitness API
| Verb   | Endpoint                    | Result                |
| ------ | --------------------------- | --------------------- |
| POST   | /api/v1/users/login         | Login User            |
| POST   | /api/v1/users/register      | Create User           |
| POST   | /api/v1/users/register/cancel      | Delete User           |
| POST   | /api/v1/users//login/forgot-password         | Login User            |
